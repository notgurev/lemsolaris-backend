## Страницы

Элементы в подсписках расположены по приоритету. Если нижние не успеваем, то не делаем. Действия/кнопки — в обратных
кавычках.

### Техническое

1. Вход в аккаунт
2. Страница ошибки

### Списки

1. Аномалии
2. Сотрудники
   1. `Выдать зарплату`
3. Полеты
4. Биржа работников
5. Корабли
6. Ресурсы
7. Отчеты о полетах

### Одиночные

1. Аномалия
    1. Вся информация
    2. Погода
    3. Ссылка на отчет (если есть)
2. Сотрудник (или фантом)
    1. Вся информация
    2. `Уволить`
    3. `Убить` (если фантом)
    4. Полеты
    5. `Создать фантома`
3. Полет
    1. Статус
    2. Дата и время начала и конца
    3. `Отменить`
    4. Аномалии
    5. Назначенные работники и ресурсы
    6. Туристы (без отдельной страницы для каждого туриста)
4. Изменение полета
5. Работник на бирже
    1. Все информация
    2. `Нанять`
6. Отчет
    1. Вся информация
    2. Ссылка на аномалию
    3. Ссылка на полет
